Instructions

Destroy all Klingon warships within the specified number of Star Dates.

The mission will end if:

  • The Enterprise destroys all Klingon warships
  • The Enterprise is destroyed by a Klingon warship
  • The Enterprise runs out of energy
  • The Enterprise runs out of time
When the mission has ended, a new one be started by clicking the 'New Game' link.

A number of Star Bases are location throughout the galaxy. Docking with a Star Base replenishes the Enterprise's energy and photon torpedoes, and also repairs all damage. The Enterprise can dock with a Star Base by positioning itself in an adjacent sector.

Parts of the Enterprise will fail spontaneously. As you navigate around they will slowly get repaired.

Commands

nav {direction: number} {distance: number}

Moves the Enterprise for a given distance in the specified direction.

Direction is expressed as a number between 1.0 and 9.0, using a polar coordinate system.

Distance is specified as a number between 0.1 and the current maximum warp factor. To navigate within the current quadrant, specify a distance < 1.

This command will fail if:

  • the direction specified is invalid
  • the distance specified is invalid
  • the distance specified exceeds the current maximum warp factor
  • insufficient energy is available to move the specified distance
  • an obstacle is blocking the way

she {add|sub} {amount: number}
Add or subtract energy from the shields. This command will fail if:
  • the shield controls have sustained damage
  • the amount specified is less than zero
  • the amount specified is greater than the ship's available energy
pha {energy: number}

Fire phasers using the specified amount of energy. Phasers will automatically target each Klingon warship in the current quadrant. Phasers are more effective at close range.

This command will fail if:

  • the phaser controls have sustained damage
  • the amount specified is less than zero
  • there are no Klingon ships in the current quadrant

pho {direction: number}

Fire a photon torpedo in the specified direction.

Direction is expressed as a number between 1.0 and 9.0, using the polar coordinate system described above.

This command will fail if:

  • the photon torpedo controls have sustained damage
  • photon torpedos have been exhausted
  • there are no Klingon ships in the current quadrant
  • the direction specified is invalid

If a photon torpedo enters a sector which contains a Star, it will be captured by the star's gravitational pull and not reach its intended target.

If a photon torpedo enters a sector which contains a Star Base, the Star Base will be destroyed and the photon torpedo will not reach its intended target.

If there are Klingons in the current quadrant, a photon torpedo command can be generated by clicking on a sector containing a Klingon warship.

nav calc {column: number} {row: number}

Calculate the command required to navigate to the specifed quadrant. If the quadrant is valid, the navigation command will be automatically inputted into the command input console.

Both Column and Row are expressed as a whole number between 1 and 8.

This command will fail if:

  • the navigation computer has sustained damage
  • either the quadrant column or row specified is not a whole number between 1 and 8
  • the quadrant specified is the current location of the Enterprise

Alternatively, a navigation command can be calculated by selecting a location on either the Short-Range or Long-Range Scanner.
